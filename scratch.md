In the administrator module, I want to have a toggle to switch the default theme of this website to something like a "hacked mode" where the website looks like it's been hacked, with a black background, green text, and some "hacker" style fonts. This would be a fun Easter egg for users to discover and could add some personality to the site. The toggle should be easy to find and use, and it should remember the user's preference for future visits.

When this is toggled, the following changes too:
1. The front page text changes from "Brewing fellowship,  one cup at a time" to "Order the world's worst drinks!"
2. "Order Here" button changes to "Get Poisoned Here"
3. Every menu item is randomly assigned prefixes. For example "ESpresso" is changed to "The Worst Espresso", "Caffe Latte" is changed to "The Most Disgusting Caffe Latte", and "Cappuccino" is changed to "The Awful Cappuccino". The prefixes should be randomly assigned each time the page is loaded in hacked mode, so that users can have a different experience each time they visit the site.
4. Customized options names can stay the same (no changes here)
5. We currently have a auto generated Super hero name for each user when they check out, but in hacked mode, we can change the set of names to hacker names. For example, instead of "Doctor Macciatto" or "Mega Fighter", users could get names like "Shadow Hacker" or "Poisonous Byte". This would add to the fun and immersive experience of the hacked mode, making it feel like users are truly stepping into a hacker's world when they visit the site.
